Select the target period you are reporting for

  • TP8 interim report from 1 January 2028

  • TP9 interim report from 1 January 2030

Select target period

Choose the preferred target period from the options that are currently available.

TP6 reporting should be done through the 'Download target period reporting (Final) spreadsheets' and 'Upload target period reporting (Final) spreadsheets' tasks.

The list only includes target periods for which you can submit a report (or a correction to a previously submitted report). Additional target periods will become available once their respective reporting periods start.
